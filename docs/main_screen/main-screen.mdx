## Меню действий с картой
*Пользователь может также получить дополнительное меню путем долгого нажатия на карту. Это основное
действие, но поведение может измениться при работе с [Публичной кадастровой картой](/pkk/pkk)*

### Оно содержит:
1. Сохранить точку в избранное
2. Добавить точку по координатам
3. Вынести точку
4. Узнать систему координат
5. Узнать высоту точки

*Все действия, требующие координаты, будут выполняться используя координаты центра карты (крест).*
