## Основные элементы:
1. [Настройки](/settings/settings) - настройки в основном относящиеся к карте
2. Поддержка - наш [чат в Telegram](https://t.me/geoeye_chat)
3. [Профиль](/profile/profile) - экран текущего пользователя
4. [Группа в Telegram](https://t.me/geo_eye)
5. Что нового - список последних обновлений
6. Ссылки - ссылки на сторонние сервисы, используемые в приложении

*На этом экране пользователь может выполнить настройку приложения под свои нужды, а также выполнить вход в свой
[профиль](/profile/profile). Зарегистрированные пользователи получают дополнительные возможности по работе с
приложением (на данный
момент - это добавление и просмотр комментариев к геопунктам)*
